常用注意

  1. 连续两个以上的空格+回车进行换行操作；
  2. Markdown只能自动识别一个半角空格，多余的空格需要使用&nbsp;或者&#160;来完成
  3. 许多语法上的特殊符号，如”* _ # + -“等，在前面加反斜线\进行转义即可显示符号本身
  4. Markdown本身不支持修改字体、字号、颜色等功能，但可以使用HTML的方法进行实现
  5. 锚点功能并不是都支持，谨慎使用
  6. latex、流程图、TODO列表等高级功能不是所有都能显示成功的，有些需要安装各自插件，这里不统一介绍，需要的请自行搜索资料
